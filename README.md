# ShopESP 2.0

A simple web-based e-commerce platform built with PHP and MySQL. This project was created as part of my college project and practicing backend.

## 🚀 Features

- User registration and login system
- Product listing and details page
- Shopping cart functionality
- Admin dashboard (basic)
- MySQL database integration

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML, CSS
- JavaScript (optional)
- XAMPP (Apache + MySQL)

## 📁 Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/shopesp2.0.git
```

2. Move the project to your htdocs folder (if using XAMPP):
```bash
C:/xampp/htdocs/shopesp2.0
```

3. Import the database:
    i. Open phpMyAdmin
    ii. Create a database (e.g., shopesp2)
    iii. Import the SQL file from /database/ folder
